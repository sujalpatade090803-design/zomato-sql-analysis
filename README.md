# Zomato SQL Business Analysis

This project simulates a food delivery database like Zomato and performs SQL-based business analysis.

## What I built
- Database schema (customers, restaurants, orders, deliveries)
- Sample data
- 50+ business queries

## Business Questions Answered
- Which restaurants generate highest revenue?
- Who are top customers?
- What is average delivery time?
- Which agents deliver fastest?
- Order & revenue trends

## Skills Used
SQL, Joins, Aggregations, CTEs, Subqueries, Window Functions, Stored Procedures
